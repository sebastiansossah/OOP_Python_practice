Title: Math printing
Description: An app that tells the user provide the start coordinates of geometrical shapes such as squares
and rectangles, their dimensions, and their colors, and the program produces an image file canvas with
the geometrical shapes drawn in it
Objets:
* Square:
        x
        y
        side
        color
        draw(canvas)
* Rectangle:
        x
        y
        width
        height
        color
        draw(canvas)
* Canvas:
        Width
        height
        color
        nake(imagepath)
